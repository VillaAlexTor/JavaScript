# HolaMundo - Ejemplos de JavaScript

Repositorio con ejemplos y ejercicios de JavaScript organizados por temas.

Estructura principal:

- `index.html` — página principal con ejemplos.
- `app.js` — script principal.
- `01-tipos/` ... `06-EjerciciosIA/` — carpetas con ejercicios y ejemplos (ver contenido).

Requisitos:

- Un navegador web moderno para abrir `index.html`.
- (Opcional) Node.js/npm para utilidades como `npx http-server`.

Cómo usar:

1. Abrir directamente en el navegador: doble clic en `index.html` o arrastrar el archivo al navegador.

2. Levantar un servidor local (recomendado para evitar problemas con rutas/fetch):

   - Con Python (si está instalado):

     ```powershell
     cd "d:\Alex\Informatica\JAVASCRIPT\HOLAMUNDO"
     python -m http.server 8000
     ```

   - Con `http-server` (npx):

     ```powershell
     cd "d:\Alex\Informatica\JAVASCRIPT\HOLAMUNDO"
     npx http-server -p 8000
     ```

   Luego abrir `http://localhost:8000` en el navegador.
